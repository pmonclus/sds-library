◆ sds_on_version_mismatch()

void sds_on_version_mismatch ( SdsVersionMismatchCallback callback)

Set callback for schema version mismatch detection (owner role only).

When an owner receives a message from a device with a different schema version, this callback is invoked. The callback can decide whether to accept or reject the message.

If no callback is registered, mismatches are logged as warnings and messages are accepted (backward compatible behavior).

Parameters
callbackFunction to call on version mismatch (NULL to use default)
See also
SdsVersionMismatchCallback, sds_get_schema_version
